Automated LLM Quiz Answering System

Endpoint for Recurrent URL Scraping and LLM Analysis

This engine utilizes a Large Language Model (LLM) combined with a **web-scraping pipeline** to automate the process of answering questions based on live web content. It's designed to handle **recurrent requests** for analysis and data extraction from any specified URL.

  • **Input:** Accepts a URL and a set of questions (or a generalized task) for analysis.
  • **Processing:** The system first **scrapes** the target URL, extracting all relevant textual content.
  • **LLM Analysis:** The extracted content is fed to the LLM, which uses the information to intelligently answer **all questions**.
  • **Output:** The final answers are formatted into a structured **JSON object** for seamless submission to the designated external endpoint.
